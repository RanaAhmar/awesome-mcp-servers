# Contribution Guidelines

Thank you for your interest in contributing to this Awesome List!

Please ensure that your pull request meets the following criteria:
1. The project must have a clear value proposition related to the Model Context Protocol (MCP).
2. The project should have an open-source license.
3. Add a concise, objective description to your entry.
4. Keep the alphabetical order within the respective categories.

Your help is greatly appreciated!
